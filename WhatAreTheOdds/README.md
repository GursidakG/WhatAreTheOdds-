# LineDesk Sports Odds Intelligence

A polished static prototype for a premium sports odds comparison product.

## Open Locally

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173
```

## Included

- Age gate
- Lobby / homepage
- Live score ticker
- Sport tabs
- Odds comparison board
- Live in-play market cards
- Best bets / value finder
- Bet tracker dashboard
- Bet slip drawer with payout calculation
- Responsive desktop and mobile layouts

This is a frontend prototype using mock data in `app.js`.
